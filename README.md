## contribeau
visualize your spotify listening history as github contributions and share with friends

note: read the disclaimer on the landing page

enjoy :)

## Tech Stack
- **Next.js 16** (App Router, Turbopack)
- **React 19**
- **Supabase** — auth & listening history storage
- **Spotify Web API** — OAuth login, recently played, top items, now playing
- **Tailwind CSS 4** + **Motion** (Framer Motion)
